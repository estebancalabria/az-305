# 📘 Programa del Curso AZ-305 – Cloud Architecture & Design

Este curso está organizado en 4 días intensivos, cubriendo los principales dominios de arquitectura en Azure: gobernanza, cómputo, datos, seguridad, redes y migración.

---

# 📅 Día 1 – Fundamentos de Arquitectura y Gobernanza

## 🧭 Cloud Adoption & Architecture
- Cloud Adoption Framework (CAF)
- Well-Architected Framework (WAF)

## 🏛️ Governance
- Suscripciones y organización de recursos
- Políticas y control de costos
- Buenas prácticas de gobernanza en Azure

## ⚙️ Compute
- Máquinas virtuales
- App Services
- Contenedores
- Escalabilidad y disponibilidad

---

# 📅 Día 2 – Datos en Azure

## 💾 Storage
- Azure Storage Accounts
- Blob, File, Table Storage
- Redundancia y performance

## 🗄️ Database
- Azure SQL Database
- Cosmos DB
- Modelos relacionales vs NoSQL
- Escalabilidad y consistencia

## 🔗 Data Integration
- ETL y ELT en Azure
- Data Factory
- Integración entre servicios de datos

---

# 📅 Día 3 – Seguridad, Mensajería y Observabilidad

## 🔐 Messaging & IoC
- Service Bus
- Event Grid
- Event Hubs
- Patrones de mensajería

## 🔑 Auth
- Microsoft Entra ID (Azure AD)
- Autenticación y autorización
- Managed Identities

## 📊 Monitoring
- Azure Monitor
- Log Analytics
- Application Insights
- Alertas y observabilidad

---

# 📅 Día 4 – Redes, Resiliencia y Migración

## 🌐 Networking
- Virtual Networks
- Subnets
- VPN Gateway / ExpressRoute
- Peering y arquitectura de red

## 🔄 Business Continuity
- Backup y Restore
- Disaster Recovery
- Alta disponibilidad

## 🚀 Migration
- Azure Migrate
- Estrategias de migración
- Lift & Shift vs modernización

---

# 🎯 Objetivo del curso

Al finalizar el curso, el estudiante será capaz de diseñar soluciones en Azure aplicando principios del:
- Cloud Adoption Framework (CAF)
- Well-Architected Framework (WAF)
- Arquitectura segura, escalable y resiliente

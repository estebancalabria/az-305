Este repositorio acompaña el curso **AZ‑305: Designing Microsoft Azure Infrastructure Solutions**, que se dicta con el objetivo de preparar de forma integral a los asistentes para diseñar soluciones de infraestructura en la nube — alineadas con los estándares de **Microsoft Azure** — y para rendir con éxito el examen de certificación oficial. :contentReference[oaicite:0]{index=0}

## 📌 ¿Qué es el AZ‑305?

El examen **AZ‑305: Designing Microsoft Azure Infrastructure Solutions** es la evaluación clave que mide tu habilidad para diseñar soluciones avanzadas en Azure que cumplan con requisitos de negocio, rendimiento, seguridad, disponibilidad y costo. Se enfoca en temas como identidad y gobernanza, monitoreo, almacenamiento de datos, continuidad del negocio e infraestructura completa. :contentReference[oaicite:1]{index=1}

Este examen es parte esencial para obtener la certificación **Microsoft Certified: Azure Solutions Architect Expert**. :contentReference[oaicite:2]{index=2}

## 🎯 Objetivos del Curso

Este curso te dará las herramientas para:

✅ Comprender los dominios y objetivos oficiales del examen AZ‑305  
✅ Diseñar soluciones de identidad, gobernanza y supervisión eficientes  
✅ Seleccionar arquitecturas óptimas de almacenamiento de datos  
✅ Implementar estrategias de alta disponibilidad y recuperación ante desastres  
✅ Diseñar soluciones de infraestructura multinivel (compute, redes, seguridad)  
✅ Traducir requisitos de negocio en arquitecturas reales en Azure

## 🧠 Temas Clave del AZ‑305 (Resumen Oficial)

El examen mide tu habilidad para *diseñar* soluciones en estos dominios: :contentReference[oaicite:4]{index=4}

1. **Diseño de soluciones de identidad, gobernanza y monitoreo** (25‑30%)  
2. **Diseño de soluciones de almacenamiento de datos** (20‑25%)  
3. **Diseño de soluciones de continuidad del negocio** (15‑20%)  
4. **Diseño de soluciones de infraestructura** (30‑35%)  

> *Cada uno de estos módulos combina teoría, mejores prácticas de arquitectura y ejercicios aplicados.*
